RedisRaft 是什么 #

RedisRaft 整体架构

RedisRaft 是 redis 社区官方使用 Raft 算法解决 redis 分布式中数据强一致问题的方案，它归属于 Redis7 的一个开发方向。 它基于 RedisModule 和 Raft 算法实现，可以灵活的插拔到 Redis7.0 以上的版本中。 图中是 RedisRaft 的架构，它基于一个 c 实现的 RaftLibrary 使用异步的 I/O 驱动的 RPC 在集群节点间 运行 Raft 算法，并且实现了持久化的 RafLog，用户的命令请求来到 RedisRaft 之后会排队提交到 Raft 线程中，它会提交请求到 Raft 状态机，当集群 复制组中半数节点以上投票之后，命令会被应用 (AppliedCommand) 到每一个节点中。
