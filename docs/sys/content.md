# 分布式键值存储系统原理-基于 ETCD 工业实现分析

本书以实践一个分布式键值存储系统为目标，由浅入深的给大家介绍了分布式系统的基本原理和实践。

从体验 ETCD raftexample 开始，你将能够直观地感受到分布式系统的运作方式。在这个过程中，你会探索分布式存储系统所面临的问题。以下是你将会逐步学习的内容：

0. [上手体验基于 Raft 算法实现的分布式 KV 服务。](https://eraft.cn/sys/ch1)

1. bbolt 存储引擎的基本原理： 了解 bbolt 的内部工作机制，包括存储结构、数据写入和读取路径，以及如何优化读写性能。

2. 分布式系统节点之间的交互远程过程调用 (RPC) 的原理： 学习节点间如何通过远程过程调用进行通信，实现数据交换和协调操作。

3. 分布式系统基本概念： 学习分布式系统的核心概念，如一致性、可用性、分区容忍性等，并理解为什么要使用这些概念来设计系统。

4. Raft 共识算法的核心概念： 深入研究Raft共识算法，包括领导者选举、日志复制、日志快照以及节点变更的过程。

5. ETCD raftexample 的实现分析： 结合实际 ETCD 案例，学习如何应用分布式一致性协议，掌握在实际系统中如何保持数据的一致性。

7. 分析 ETCD 分布式键值存储系统设计与实现，加深对分布式系统设计的理解。

8. 分布式系统的横向扩展和多分片： 探索如何通过多分片来横向扩展 ETCD，以支持处理海量数据的存储需求。

9. 分布式事务的基础知识： 了解分布式事务的挑战和实现方式，为设计更复杂的分布式系统打下基础。

通过以上逐步的学习，你将逐渐深入了解分布式系统的各个方面，从基础原理到实际应用，为你掌握设计和开发分布式存储系统的技能提供坚实的基础。
